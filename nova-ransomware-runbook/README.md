# NOVA Ransomware – SOC Runbook Pack

This repository contains a one-page SOC runbook, RACI, workflow diagram, compliance mappings (ISO 27001 / NIST / SOC 2), and a vendor security addendum clause for NOVA ransomware incidents.

## Contents
- `runbooks/ransomware-nova-one-page.md` — one-page SOC runbook (M365 + Google Workspace procedures included)
- `governance/nova-ransomware-raci.md` — RACI matrix
- `diagrams/nova-ransomware-workflow.mmd` — Mermaid workflow diagram
- `governance/control-mapping-iso-nist-soc2.md` — control mapping (ISO 27001 / NIST CSF & 800-53 / SOC 2)
- `vendor/vendor-security-addendum-nova-ransomware.md` — contract-ready clause

## How to render the diagram
GitHub supports Mermaid in Markdown. You can paste the contents of `.mmd` into a Markdown mermaid block or use a Mermaid preview extension.
